What is this?

This app demonstrates how an AI learns using trial and error.

  • Tries random actions
  • Learns from results
  • Improves automatically

This process is called Neuroevolution.

Technical Details

Population Size

Number of agents tested each generation.

Typical Values: 20 - 100

Mutation Strength

Controls randomness added to weights.

Typical Values: 0.05 - 0.2

Learning Rate

Controls speed of learning.

Typical Values: 0.05 - 0.2

Hidden Layer Size

Number of neurons in neural network.

Typical Values: 4 - 16

Pseudocode

Initialize weights randomly 



For each generation: 

     Create population 

     Run agents 

     Collect rewards 

     Select best agents 

     Update weights 



Repeat
